DDL - Data Definition Language:

Engloba os comandos para definir um banco de dados:

- CREATE : Cria objetos, por exemplo Banco de Dados, Tabelas, etc.
- DROP : Exclui objetos, por exemplo, tables, databases, etc.
- ALTER : Altera objetos existentes, por exemplo, ALTER TABLE.
- TRUNCATE : Exclui todo conteúdo de uma tabela e redefine a mesma com o seu valor incial.
- COMMENT : Comentários, -- ou /* */

-------------//-------------

DQL - Data Query Language:

Responsável por comandos de consulta aos dados:

SELECT : Utilizado para consultar dados em uma tabela

-------------//-------------

DML - Data Manipulation Language:

Utilizado para manipulação de dados:

INSERT: Insere dados em uma ou mais tabelas
UPDATE: Atualiza os dados já existentes em uma tabela ou mais tabelas
DELETE: Exclui os registros de uma tabela ou mais.

-------------//-------------

DCL - Data Control Language

Relacionado à segurança do banco de dados:

GRANT: Fornece a um determinado usuário o privilégio de acesso do banco de dados
REVOKE: Retira os privilégios de acesso de um determinado usuário

-------------//-------------

DTL - Data Transaction Language

Relacionado às transações de dados

BEGIN TRANSACTION: Inicia um processo de transação para a sessão atual.
COMMIT: Finaliza o processo de transação atual na sessão, confirmando todas as alterações efetuadas desde o início do processo.
ROLLBACK: Finaliza o processo da transação atual, descartando todas as alterações efetuadas desde o início do processo
